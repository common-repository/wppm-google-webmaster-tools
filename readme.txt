== Description ==   

WPPM Google Webmaster Tools help you to appearing in search results.

Google Webmaster Tools (GWT) is the primary mechanism for Google to communicate with webmasters. 

<h4>Features</h4>
    
* Easy to Install and use
* Easy to Configure Google Webmaster Tools(GWT)
* Easy to Bing Webmaster Tools
* Easy to Alexa Verification Tools

== Installation ==
1. Upload the entire `wppm-google-webmaster-tools` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'Google Webmaster Tools' in WordPress.
3. Add your Google Webmaster Tools Verification Tracking code to the Google Webmaster Tools settings.
